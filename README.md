## Installation & Setup
1. **Clone the repository:**  
   ```sh
   git clone https://github.com/Eshwarsai-07/scatch.git
   ```
2. **Navigate into the project folder:**  
   ```sh
   cd scatch
   ```
3. **Install dependencies:**  
   ```sh
   npm install
   ```
4. **Create a `.env` file** and add the necessary environment variables.
5. **Start the server:**  
   ```sh
   npm start
   ```
